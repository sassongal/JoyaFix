# JoyaFix 🇮🇱

אפליקציה מקצועית למק עם ניהול לוח העתקה וקיצור דרך להמרת עברית/אנגלית.

## תכונות עיקריות

### 1. המרת טקסט חכמה (Hebrew ↔ English)
- **קיצור דרך גלובלי**: לחץ על המקש המוגדר (ברירת מחדל: Cmd+Option+K)
- **זיהוי אוטומטי**: מזהה אוטומטית אם הטקסט הוקלד במקלדת הלא נכונה
- **המרה מיידית**: מעתיק, ממיר ומדביק אוטומטית
- **תמיכה ב-RTF**: שומר עיצוב (צבעים, פונטים, בולד וכו')
- **צליל הצלחה**: אופציונלי - צליל כשההמרה הצליחה

### 2. מנהל היסטוריית לוח
- **20 פריטים אחרונים** (ניתן לשינוי בהגדרות)
- **תמיכה ב-Rich Text**: שומר עיצוב מלא מ-Pages, Word, אתרי אינטרנט
- **נעיצת פריטים**: נעץ פריטים חשובים למעלה
- **קיצורי מקלדת**: Cmd+1 עד Cmd+9 לגישה מהירה
- **חיפוש והדבקה**: לחץ על פריט כדי להדביק
- **שמירה אוטומטית**: ההיסטוריה נשמרת בין הפעלות

### 3. חלון הגדרות מלא
- **שינוי קיצור דרך**: בחר כל שילוב מקשים שתרצה
- **כמות היסטוריה**: קבע כמה פריטים לשמור (5-100)
- **התנהגות**: בחר אם להדביק אוטומטית ולהשמיע צליל
- **איפוס**: חזור להגדרות ברירת מחדל בלחיצה

## דרישות מערכת

- macOS 11.0 (Big Sur) ומעלה
- הרשאות Accessibility (נדרש לסימולציה של מקשים)

## התקנה ובנייה

### שלב 2: הרצת האפליקציה

```bash
open build/JoyaFix.app
```

### שלב 3: מתן הרשאות

1. פתח **System Preferences → Security & Privacy → Privacy → Accessibility**
2. לחץ על המנעול כדי לבצע שינויים
3. הוסף את **JoyaFix** לרשימה
4. סמן את התיבה ליד JoyaFix

## שימוש

### המרת טקסט
1. בחר טקסט כלשהו (בכל אפליקציה)
2. לחץ **Cmd+Option+K** (או המקש שהגדרת)
3. הטקסט יומר ויודבק אוטומטית!

**דוגמאות:**
- `shalom` → `שלום`
- `שלום` → `shalom`
- `vhuo` (הוקלד במקלדת עברית) → `shalom`

### שימוש בהיסטוריה
1. לחץ על אייקון **א/A** בשורת התפריטים
2. בחר פריט מהרשימה
3. הפריט יודבק אוטומטית

**קיצורי מקלדת:**
- `Cmd+1` - פריט ראשון
- `Cmd+2` - פריט שני
- וכן הלאה עד Cmd+9

### נעיצת פריטים
1. לחץ על פריט בהיסטוריה
2. בחר **Pin**
3. הפריט יישאר תמיד בראש הרשימה 📌

### הגדרות
1. לחץ על אייקון **א/A** בשורת התפריטים
2. בחר **Settings...**
3. שנה הגדרות לפי הצורך

## פתרון בעיות

### האפליקציה לא עובדת
- וודא שנתת הרשאות Accessibility
- בדוק את Console.app עבור הודעות שגיאה

### ההמרה לא עובדת
- ודא שהקיצור דרך לא תפוס על ידי אפליקציה אחרת
- נסה לשנות את הקיצור דרך בהגדרות

### ההיסטוריה לא נשמרת
- בדוק שיש הרשאות כתיבה ל-UserDefaults
- נסה לאפס את ההגדרות
